# Dilepton Production as a Probe for Pion Condensation in Hot and Dense QCD Matter

This repository contains the datafiles used to generate 
Figures 2–10 in the paper:

**Aritra Bandyopadhyay**, *Dilepton production as a probe for pion condensation in hot and dense QCD matter*,
arXiv:2604.21633 [hep-ph] (2026)
https://arxiv.org/abs/2604.21633

## Data Description
Each datafile corresponds to a specific figure in the paper. The column names and units are described in the respective figure captions of the paper.

## Usage
The data can be used to reproduce the figures in the paper. Readers are encouraged to cite the paper if this data is used in their work.
